Why you can trust it

Four gates. One independent critic.

A fix isn’t “done” because the code looks right. It’s done when it survives all of these — every round, no exceptions.

GATE 1

axe re-scan

The violation is gone, total is lower, nothing new introduced.

GATE 2

pixel diff

Zero pixels changed outside masks — the design is untouched.

GATE 3

banned patterns

No aria-hidden, alt="", display:none — no suppression tricks.

GATE 4

IBM Equal Access

A genuinely independent engine agrees. Not Lighthouse — that’s axe again.

THE CRITIC · independent model on Akash

Semantic truth — is the fix actually right?

Axe checks an alt attribute exists. Only something that can see the image checks it’s true. The critic runs on a different model family, on a different provider, so the fixer can never grade its own homework. Two axes of independence: engine and model.
